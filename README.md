Spring Boot Back End School Project

📜 Description
The project is a back-end for a vacation excursion booking application. The back-end is built with Java and the Spring Boot framework, and it connects to a MySQL database to persist data. It exposes a series of RESTful endpoints for the front-end to consume, allowing users to view and purchase vacation packages.

🛠️ Technologies Used
Java: The core programming language for the application logic.

Spring Boot: The framework used to build the RESTful API and manage dependencies. 

MySQL: The relational database used for data storage. 

JPA (Java Persistence API): Used for object-relational mapping between Java objects and the database.

Maven: The build automation and dependency management tool. 

Lombok: A library to reduce boilerplate code for model/data objects. 

Git: For version control. 

✨ Features
RESTful API for managing vacations, excursions, and customers.

CRUD (Create, Read, Update, Delete) operations for all major entities.

A shopping cart feature allowing customers to add and remove vacation packages.

A checkout process to finalize a purchase.
